# Demo Workspace

This folder contains the canonical demo content for Sly screenshots and launch materials.

It includes:

- a realistic folder structure
- sample notes with believable previews
- `.sly/tasks.db` with demo tasks across Inbox, Today, Upcoming, Waiting, Anytime, Someday, Starred, and Completed
- tagged tasks for release, design, screenshots, writing, field notes, recurring work, and waiting items
- `.sly/settings.json` so folder icons, task tag icons, colors, pinned notes, recent notes, collapse state, and note-list preferences match the intended demo state

## Recommended Notes For Screenshots

- `Field Notes/Sly profile.md`
- `Product/Launch/1.0 screenshot checklist.md`
- `Product/Launch/Task tag setup.md`

## Usage

Copy this workspace to a local folder and point Sly at it when you need a clean, reproducible demo setup.
