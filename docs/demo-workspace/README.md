# Demo Workspace

This folder contains the canonical demo content for Sly screenshots and launch materials.

It includes:

- a realistic folder structure
- sample notes with believable previews
- `.sly/settings.json` so folder icons, folder colors, pinned notes, and recent notes match the intended demo state

## Recommended Notes For Screenshots

- `Field Notes/Sly profile.md`
- `Product/Launch/1.0 screenshot checklist.md`

## Usage

Copy this workspace to a local folder and point Sly at it when you need a clean, reproducible demo setup.
