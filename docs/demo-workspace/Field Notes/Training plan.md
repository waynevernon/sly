# Training plan

The goal is not more tricks. The goal is steadier attention in distracting places.

## Current focus

- settle quickly after excitement
- tighten up the recall finish
- make loose-leash walking feel automatic again

## Simple structure

- Monday / Wednesday / Friday: short obedience reps
- Tuesday / Thursday: longer sniff walk with one or two clean recalls
- Weekend: one outing somewhere mildly busy, then stop while it still feels easy
