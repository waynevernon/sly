# Recall cues

Use the same wording every time. Clarity matters more than variety.

| Cue | Meaning | Reminder |
| --- | --- | --- |
| Here | Return all the way in | Reward at the finish, not halfway |
| Close | Come into heel position | Keep it brief and clean |
| Break | You are released | Use it consistently |

If a rep feels muddy, reset and make the next one easier.
